## Clean Installation

1. Boot into **Recovery**.
2. Perform the following wipes:

   * Data
   * Metadata
   * Dalvik / ART Cache
   * Cache
3. Flash the **ROM.zip**.
4. Flash **GApps** *(only if using Vanilla build — NikGapps Core recommended)*.
5. Go to **Format Data** and type `yes`.
6. Convert partitions to **F2FS (MANDATORY)**:

   * Go to **Wipe → Advanced Wipe**
   * Select **Data** → **Repair or Change File System** → **Change File System → F2FS** → Swipe to confirm
   * Repeat the same steps for **Cache**
7. Reboot to **System**.

---

## Update (Dirty Flash)

> [!NOTE]
> Dirty flashing may not work across **major Android version upgrades**.

1. Flash the **ROM.zip**.
2. Flash **GApps** again *(only if previously using Vanilla + GApps)*.
3. Wipe:

   * Dalvik / ART Cache
   * Cache
4. Reboot to **System**.

---

## OTA Update

1. Go to **Settings → System → System Updates**.
2. Download the latest build.
3. Tap **Reboot** after download completes.
4. Device will automatically install the update.
5. Reboot to **System**.

---

> [!IMPORTANT]
> If you are using a **Vanilla build**, you must **reflash GApps after every update**,
> including **OTA** and **dirty flash**.
